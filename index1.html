📘 Theory & docs

▪️ components: accumulator (E₀ variable), Leclanché (E₁), Daniell (E₂), galvanometer, two‑way switch, keys, 5kΩ resistor, resistance box, jockey, wire 1‑4m.

Figure 33.1 + resistance box

📐 Theory
If l₁ balance for E₁ and l₂ for E₂, then E₁/E₂ = l₁/l₂. For graphical method, vary R and plot l₁ vs l₂ → slope = E₁/E₂. E₀ (accumulator) can be adjusted.
🔧 Procedure
  1. Set up circuit, close K₁, K₂ open.
  2. Connect two‑way to E₁. Touch A & B: opposite deflections = correct.
  3. Find rough balance (K₂ open), then accurate (K₂ closed). Measure l₁.
  4. Repeat for E₂ → l₂.
  5. For graphical: insert resistance box, record (l₁,l₂) for different R.
  6. Plot l₁ vs l₂, slope gives E₁/E₂.
💬 Discussion & precautions
  • E₀ must exceed E₁ and E₂ (adjust with slider).
  • Opposite deflections at A & B confirm polarity; otherwise reverse or check connections.
  • Use 5kΩ resistor to protect galvanometer.
  • Clean jockey and wire.
  • Graphical method minimises random errors.
  • If no deflection, check for loose wires or discharged E₀.
