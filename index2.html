Selected: Dry cell (E=1.52V, r=0.82Ω)

📐 Theory

For a cell with emf E and internal resistance r, terminal voltage V = E·R/(R+r).

Potentiometer: V = K·l → 1/l = (K·r/E)(1/R) + K/E

r = gradient / intercept (from 1/l vs 1/R plot)

🔬 Procedure

  1. Set up circuit as in circuit.PNG (Figure 34.1)
  2. Select battery and set R = 50Ω
  3. Find balance point by sliding jockey (click on wire)
  4. Record R and l in table
  5. Reduce R by 5Ω, repeat until 20Ω
  6. Plot graph and compute r

💬 Discussion

The measured r should match the battery's true internal resistance. The centre-zero galvanometer increases accuracy. Different battery types show different internal resistances.
